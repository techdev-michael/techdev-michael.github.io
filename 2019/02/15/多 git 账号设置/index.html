多个 Github 账号设置

发表于 2019-02-15 | 阅读次数：

在实际开发过程中，我们可能在一台机器上登录两个 git 账号，如一个公司分配的账号以及自己在 Github上注册的账号，此时，如果我们向远程仓库提交代码时，则需要区分开不同的账号执行相应的操作，具体如何配置的，我们接下来一步步实现（windows 机器，其他平台同理）.

具体场景

假设我们已经在机器上生成并在远程仓库设置好了公司账号的 SSH 秘钥，已经可以顺利向远程库提交代码，接下来要让第二个账号也能随心所欲的实现代码提交到另一个远程仓库的操作。
